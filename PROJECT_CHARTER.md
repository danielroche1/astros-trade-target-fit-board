# Astros Left-Handed Outfield Trade Target Board

## Baseball Question
Which realistic left-handed or switch-hitting outfield trade targets best improve Houston’s lineup balance and production vs RHP while preserving defensive flexibility and manageable acquisition cost?

## Audience
Assistant GM, Director of Baseball Operations, Pro Scouting Director

## Decision
Should the Astros pursue an external left-handed outfield bat, and if so, which targets best fit the current roster?

## Output
1. Ranked target board
2. Player cards for top targets
3. One-page recommendation memo
4. Public-facing portfolio version

## Core Evaluation Buckets
- Offensive fit vs RHP
- Underlying Statcast quality
- Defensive / positional fit
- Baserunning / athletic value
- Contract / control
- Risk
- Market realism

## Limits
- Public data only
- Trade availability is estimated, not known
- Health and internal scouting information are incomplete
